# Интерфейс Set и класс HashSet (Java)

## 1. Что такое Set
- `Set` — это интерфейс коллекций, представляющий **множество уникальных элементов**.
- Основные характеристики:
  - Элементы **неупорядоченные** (порядок не гарантируется).
  - **Дубликаты запрещены**.
- Программирование через интерфейс:
```java
Set<String> names = new HashSet<>();
````

## 2. Класс HashSet

* `HashSet` — реализация интерфейса `Set` на основе **хэш-таблицы**.
* Быстрый доступ к элементам в среднем за O(1).
* Основные методы:

  * `add(E e)` — добавить элемент.
  * `remove(Object o)` — удалить элемент.
  * `contains(Object o)` — проверить наличие элемента.
  * `size()` — количество элементов.
  * `isEmpty()` — проверить пустоту множества.
  * `clear()` — очистить множество.

## 3. Пример использования

```java
import java.util.HashSet;
import java.util.Set;

public class Example {
    public static void main(String[] args) {
        Set<Integer> numbers = new HashSet<>();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        numbers.add(20); // не добавится, так как дубликат
        
        System.out.println("Элементы множества: " + numbers);

        if(numbers.contains(20)) {
            System.out.println("Множество содержит число 20");
        }

        numbers.remove(10);
        System.out.println("После удаления 10: " + numbers);
    }
}
```

## 4. Особенности

* `HashSet` **не сохраняет порядок элементов**.
* Если нужен **сортированный порядок**, используйте `TreeSet`.
* Для **упорядоченного добавления** можно использовать `LinkedHashSet`.
