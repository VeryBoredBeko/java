## **Лёгкая сложность (10 заданий)**

1. Создать `HashSet` целых чисел, добавить в него числа 1, 2, 3, 4, 5 и вывести на экран.
2. Создать `HashSet` строк, добавить три имени и проверить, содержит ли множество имя `"Алия"`.
3. Создать `HashSet`, добавить несколько дубликатов и вывести множество. Объяснить, что произошло.
4. Создать `HashSet<Integer>` и проверить, пустое ли множество перед добавлением элементов.
5. Удалить элемент из `HashSet` и вывести результат.
6. Создать `HashSet` и добавить 5 элементов. Использовать метод `size()` для определения количества элементов.
7. Создать `HashSet<String>` и очистить его с помощью метода `clear()`. Проверить, пустое ли множество после очистки.
8. Использовать цикл `for-each` для вывода всех элементов `HashSet` строк.
9. Создать `HashSet<Integer>`, добавить элементы и проверить наличие элемента с помощью `contains()`.
10. Создать два `HashSet` и объединить их с помощью метода `addAll()`.

## **Средняя сложность (5 заданий)**

1. Создать `HashSet<Integer>`, добавить числа от 1 до 10. Удалить все чётные числа с помощью `Iterator`.
2. Создать `HashSet<String>` и добавить несколько слов. Вывести слова в верхнем регистре с использованием цикла `for-each`.
3. Написать метод, который принимает `HashSet<Integer>` и возвращает сумму всех элементов множества.
4. Создать `HashSet<Integer>`, добавить числа 1–10, а затем создать новый `HashSet`, содержащий только числа больше 5.
5. Написать программу, которая проверяет, есть ли пересечение между двумя `HashSet` строк (использовать `retainAll()`).

## **Сложная сложность (5 заданий)**

1. Создать `HashSet<Integer>` из случайных чисел (10–50 элементов), а затем вывести элементы в порядке возрастания (использовать `TreeSet` для сортировки).
2. Написать метод, который принимает `HashSet<String>` и возвращает новое множество с длинами всех строк.
3. Создать программу, которая подсчитывает количество уникальных слов в предложении, введённом пользователем.
4. Реализовать метод, который принимает два `HashSet<Integer>` и возвращает `true`, если одно множество является подмножеством другого.
5. Написать программу, которая имитирует простую регистрацию пользователей. Хранить логины в `HashSet<String>`. Если пользователь пытается зарегистрировать уже существующий логин — вывести сообщение об ошибке.
